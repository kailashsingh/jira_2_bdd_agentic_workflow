QUICK START GUIDE:
==================

1. Prerequisites:
   - Node.js (v16 or higher)
   - npm (comes with Node.js)
   - Angular CLI (install with: npm install -g @angular/cli)

2. Setup:
   - Windows: Double-click setup.bat
   - Linux/Mac: Run ./setup.sh (make it executable first: chmod +x setup.sh)
   - Manual: Run 'npm install' in the project directory

3. Development:
   - Run: npm start (or ng serve)
   - Open: http://localhost:4200

4. Build for production:
   - Run: npm run build
   - Files will be in: dist/bdd-agentic-workflow/

CONVERSION NOTES:
=================
This project is a complete Angular conversion of the original React BDD Agentic Workflow application.

Key changes made:
- React components → Angular standalone components
- JSX templates → Angular templates with directives
- React hooks → Angular lifecycle methods
- shadcn/ui → Angular Material
- Lucide React → Material Icons
- CSS modules → SCSS with utility classes

The functionality and visual design remain identical to the original React version.
